Τα drones της Σοφίας δεν είναι άψυχα και ανόητα, έχουν τη δυνατότητα να κάνουν νέους φίλους. Για την ακρίβεια ήδη εργάζονται πάνω στη δημιουργία ενός κοινωνικού δικτύου μόνο για drones! Η Σοφία έχει λάβει τα δεδομένα για τις συνδέσεις ανάμεσα στα drones και θέλει να μάθει περισσότερα για τις σχέσεις που αναπτύσσονται μεταξύ τους.

Έχουμε έναν πίνακα που περιέχει τις απευθείας συνδέσεις μεταξύ των drones. Κάθε σύνδεση αναπαρίσταται ως String που περιέχει τα ονόματα των δύο φίλων, χωρισμένα με παύλα. Για παράδειγμα: το "dr101-mr99" σημαίνει ότι το dr101 και το mr99 είναι φίλοι. Σκοπός είναι να γράψεις μια συνάρτηση που θα επιτρέπει την ανακάλυψη πιο πολύπλοκων συνδέσεων ανάμεσα στα drones. Σου δίνονται δύο ονόματα. Προσπάθησε να αποφασίσεις αν συνδέονται μεταξύ τους, μέσω κοινών φίλων σε οποιοδήποτε βάθος. Για παράδειγμα: αν 2 drones έχουν κοινούς φίλους ή φίλους που έχουν κοινούς φίλους κοκ.

Ας δούμε μερικά παραδείγματα:
το scout2 και το scout3 έχουν σαν κοινό φίλο το scout1 οπότε συνδέονται. Το super και το scout2 συνδέονται μέσω των sscout, scout4 και scout1. Αλλά το dr101 και το sscout δε συνδέονται μεταξύ τους.

Είσοδος: Τρεις παράμετροι: Πληροφορίες για τους φίλους σε μορφή πλειάδας από strings, το πρώτο όνομα σε μορφή string, το δεύτερο όνομα σε μορφή string.

Έξοδος: Έχουν τα δύο drones κάποια σχέση μεταξύ τους ή όχι σε μορφή λογικής τιμής.

Παράδειγμα: 

check_connection(
    ("dr101-mr99", "mr99-out00", "dr101-out00", "scout1-scout2",
     "scout3-scout1", "scout1-scout4", "scout4-sscout", "sscout-super"),
    "scout2", "scout3") == True
check_connection(
    ("dr101-mr99", "mr99-out00", "dr101-out00", "scout1-scout2",
     "scout3-scout1", "scout1-scout4", "scout4-sscout", "sscout-super"),
    "dr101", "sscout") == False

Πώς χρησιμοποιείται: Η αποστολή αυτή θα σε βοηθήσει να βρίσκεις μη προφανείς συνδέσεις σε ένα δίκτυο δεσμών και να εργάζεσαι με κοινωνικά δίκτυα.

Προϋποθέσεις: len(δίκτυο) ≤ 45
Αν "όνομα1 - όνομα2" στο δίκτυο, τότε το "όνομα2- όνομα1" δεν είναι στο δίκτυο
3 ≤ len(όνομα drone) ≤ 6
Το πρώτο και το δεύτερο όνομα είναι στο δίκτυο.
